Rudimentary Grade Tracker

How to Use This

To add an assignment, fill in all three fields and submit.

To edit an assignment, use the name of an existing assignment and change the scores.

To delete an assignment, use the name of an existing assignment and leave all other fields blank.

What is this?

This tool lets you calculate your grade, weighted by the scores of each individual assignment (i.e. 100 points is worth more than 10).

The percent score for each assignment and overall is calculated on the server--isn't that nifty? This really is the future of cloud computing.
